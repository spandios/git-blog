# Aggregate

* 엔티티와 밸류 타입의 객체들을 묶어 상위 수준에서 도메인 모델 간의 관계를 파악할 수 있게 하는 것
* 애그리거트에 속한 객체는 유사하거나 동일한 라이프 사이클을 갖는다.
* 애그리거트에 속한 어느 한 객체는 다른 애그리거트에 속하지 않는다.
* 보통 애그리거트가 한 개의 엔티티 객체만 갖는 경우가 많다고 한다.
